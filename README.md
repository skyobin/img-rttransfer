# img-rttransfer
 
   本项目的应用场景为线上提供实时转图图片，包括：大小、等比压缩、裁剪、定高压缩、定宽压缩和实时转换为webpg；
从而减少线上图片存储空间并灵活适应对图片需求的变更。
